# PokeType

A simple mathematical expression evaluator that uses Pokemon types to replace symbols.

## Syntax

### Reference

| Symbol | Math Equivalent | Pokemon Type |
|--------|-----------------|--------------|
| **♨**  | +               |    Fire      |
| **≋**  | -               |    Water     |
| **ᨏ** | *               |    Rock      |
| **༄** | /               |    Flying    |
| **☇**  | //              |    Electric  |
| **☠**  | * 0             |    Poison    |
| **❅()**| abs()           |    Ice       |
| **☯**  | >               |   Fighting   |
| **❂**  | <               |    Psychic   |
| **✴**  | ==              |    Normal    |
| **❤**  | (               |    Fairy     |
| **☽**  | )               |    Dark      |
| **⸙**  | \|              |    Grass     |
| **中**  | >>>             |    Dragon    |
| **⏚**  | <<<             |    Ground   |

### Types Unused

- Steel
- Bug
- Ghost

### Data Types

- Numbers
- Boolean: "true" or "false"
