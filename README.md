# PokeType

A simple mathematical expression evaluator that uses Pokemon types to replace symbols.

## Syntax

### Reference

| Symbol | sath Equivalent | Pokemon Type |
|--------|-----------------|--------------|
| **🔥**    | +               |    Fire      |
| **🌊**    | -               |    Water     |
| **🪨**    | *               |    Rock      |
| **💨**    | /               |    Flying    |
| **⚡**     | //              |    Electric  |
| **☠️**    | %               |    Poison    |
| **✊**    | >               |   Fighting   |
| **🔮**    | <               |    Psychic   |
| **✨**    | ==              |    Normal    |
| **👻**    | != 			  |     Ghost 	 |
| **🧚**    | (               |    Fairy     |
| **🌌**    | )               |    Dark      |
| **❄️()**  | abs()           |    Ice       |
| **⛓()**  | log()			  | 	Steel    |
| **🌿**    | \|              |    Grass     |
| **🐉**    | >>>             |    Dragon    |
| **🏝️**    | <<<             |    Ground    |
| **🐛()**  | ^()			  |     Bug      |
